# Change CSF Layer Thickness

The CSF layer in Autovalidate is not segmented directly — it is the space between the brain and the skull. The skull is generated by dilating the combined brain+CSF mask outward, so the thickness of that dilation controls how much room is left for CSF.

This tutorial shows how to adjust that thickness.

## How the skull is built

In `autovalidate/morphology/skull.py`, `create_skull` dilates the combined brain+CSF mask using a spherical structuring element:

```python
from skimage.morphology import ball
from scipy.ndimage import binary_dilation

struct = ball(3)   # ← radius in voxels
outer  = binary_dilation(combined, structure=struct)
skull  = outer & ~combined
```

The radius of `ball(r)` controls the skull shell thickness, which in turn sets the gap available for CSF between the brain surface and the skull inner wall.

At the default voxel size of 1 mm³, `ball(3)` produces a shell roughly 3 mm thick.

## Changing the radius

Open `autovalidate/morphology/skull.py` and change the radius passed to `ball`:

```python
struct = ball(5)   # thicker skull shell → more CSF room
```

| `ball` radius | Approximate shell thickness |
|---------------|-----------------------------|
| 2 | ~2 mm |
| 3 | ~3 mm (default) |
| 5 | ~5 mm |

Larger values produce a thicker skull and more CSF, but also risk the shell overlapping neighbouring anatomy.

## Making the radius configurable

If you want to expose this as a parameter without editing source code, add a field to `config.toml` and thread it through the pipeline.

**1. Add the field to `config.toml`:**

```toml
[run]
skull_dilation_radius = 3   # voxels
```

**2. Add it to the `Config` dataclass in `autovalidate/config.py`:**

```python
skull_dilation_radius: int = 3
```

And parse it in `parse_config`:

```python
skull_dilation_radius = raw_dict["run"].get("skull_dilation_radius", 3)
```

**3. Update `create_skull` to accept the radius:**

```python
def create_skull(brain_mask, csf_mask, dilation_radius: int = 3):
    struct = ball(dilation_radius)
    ...
```

**4. Pass it from `run.py`:**

```python
skull_mask = skull.create_skull(brain_mask, csf_mask, cfg.skull_dilation_radius)
```

## Verifying the result

Load the output `.nii.gz` in a viewer such as ITK-SNAP or with nibabel and check the skull shell thickness visually:

```python
import nibabel as nib
import numpy as np

img = nib.load("subject_combined_labels.nii.gz")
labels = img.get_fdata()

skull_voxels = (labels == 4).sum()
print(f"Skull voxels: {skull_voxels}")
```

Adjust the radius until the shell matches the anatomy in the T1 image.
